## 1. 포지셔닝 (Positioning)

### 1.1 포지셔닝 스테이트먼트

**대상 고객 (Target):**
소규모 오프라인 매장 점주 (카페, 음식점, 소매점 등)

**고객의 문제 (Problem):**
채용 실수, 과다한 인건비, 직원 교육 부담, 법규 리스크로 인한 경영 실패

**솔루션 카테고리 (Category):**
AI 기반 자영업 HR 베스트 프랙티스 SaaS

**핵심 차별점 (Key Benefit):**
성공 매장의 HR 운영 공식을 데이터 기반으로 자동 복제하여,
누구나 따라 하면 성과가 나는 경영 시스템 제공

**경쟁 대비 우위 (Reason to Believe):**
- 단순 기록 툴이 아닌 AI 기반 성장 시스템
- Freemium→Subscription→Matching 다층 BM으로 락인 강화
- 점주·알바생·전문가 3자 확산 네트워크 효과

## 2. 경쟁 분석 (Competitive Landscape)

### 2.1 주요 경쟁사 비교

| 구분 | 샤플 (Shapl) | 시프티 (Shifty) | BestPractice HR |
|------|-------------|----------------|----------------|
| **핵심 기능** | 근태·급여 관리 | 스케줄링·커뮤니케이션 | AI 베스트 프랙티스 복제 |
| **가치 제안** | 근태 자동화 | 효율적 근무 관리 | 성공 매장 운영 공식 SaaS화 |
| **사용자 체감** | 기록 중심 | 커뮤니케이션 중심 | 성장 중심 (AI 추천·피드백) |
| **BM** | 단일 구독 | 단일 구독 | 다층형 (Free→Matching→Data) |
| **확산 전략** | 점주 직접 세일즈 | 점주 직접 세일즈 | 3자 네트워크 (점주·알바·파트너) |
| **데이터 활용** | 제한적 | 제한적 | 베스트 프랙티스 학습·모델링 |
| **브랜드** | 효율 툴 | 협업 툴 | 성공 복제 시스템 |

### 2.2 차별점 (Differentiation)

**1. "기능 툴"이 아닌 "성공 모델 복제 시스템"**
- 경쟁사: 근태/스케줄 기록·관리
- 우리: 성공 매장 데이터 학습 → AI 기반 자동 추천

**2. 다층형 BM으로 락인 강화**
- 경쟁사: 단일 구독 모델
- 우리: Free (데이터 수집) → Subscription (ROI 증명) → Matching (고마진)

**3. 3자 확산 네트워크**
- 경쟁사: 점주 중심 마케팅
- 우리: 점주·알바생·전문가 파트너 동시 확산

**4. 데이터 플라이휠**
- 경쟁사: 고립된 데이터
- 우리: 사용할수록 똑똑해지는 AI 모델 (Network Effect)

## 3. 비즈니스 모델 (Business Model)

### 3.1 5-Layer BM 구조

| 레이어 | 기능 | 타겟 | 수익 모델 | 전략적 목적 |
|--------|------|------|-----------|-----------|
| **① Free Layer** | 5분 매뉴얼<br>기본 근태<br>교육 자동화 | 모든 점주 | Freemium | 데이터 확보<br>시장 확산 |
| **② Subscription Layer** | AI 법규 준수<br>AI 스케줄링<br>효율 분석 대시보드 | 성장 지향 점주 | 월 ₩29,000~₩99,000 | ROI 기반 구독 락인 |
| **③ Best Practice Layer** | 성공 매장 모델 템플릿<br>업종별 맞춤 추천<br>실시간 벤치마킹 | 프리미엄 점주 | 월 ₩149,000~₩299,000<br>(성과 연동형 옵션) | 고단가 전환<br>차별화 핵심 |
| **④ Matching Layer** | 검증 인재 추천<br>AI 경력 코칭<br>재고용 매칭 | 모든 점주 | 채용 성공 시 수수료<br>(월급의 30~50%) | 고마진 매출 축 |
| **⑤ Data Layer** | 업종별 HR 지표 리포트<br>벤치마크 API<br>산업 인사이트 | 파트너·대기업 | 데이터 라이선스<br>API 사용료 | B2B 확장<br>산업 표준화 |

---

### 3.2 수익 흐름 (Revenue Streams)

**주요 수익원:**
1. **구독 수익 (Recurring Revenue):**
   - Standard: ₩29,000/월
   - Premium: ₩99,000/월
   - Best Practice: ₩149,000~₩299,000/월
   - 목표 ARPU: ₩85,000/월

2. **매칭 수수료 (Transaction Revenue):**
   - 채용 성공 시 월급의 30~50%
   - 평균 수수료: ₩600,000/건

3. **데이터 라이선스 (B2B Revenue):**
   - API 사용료: ₩2,000,000~₩10,000,000/월
   - 산업 리포트: ₩5,000,000~₩20,000,000/년

**수익 구조 목표 (3년 후):**
- 구독 수익: 60%
- 매칭 수수료: 30%
- 데이터 라이선스: 10%

---

### 3.3 고객 획득 전략 (Go-to-Market)

**Phase 1 (0-12개월): Freemium 확산**
- 무료 5분 매뉴얼 배포 (SEO/SNS)
- 노무사/세무사 파트너 네트워크 구축
- 알바생 커뮤니티 마케팅

**Phase 2 (13-24개월): Subscription 전환**
- ROI 증명 케이스 스터디 확산
- AI 기능 강화 (스케줄링, 법규 준수)
- 프리미엄 구독 유도

**Phase 3 (25-36개월): Matching & Data**
- 검증 인재 Pool 활성화
- 업종별 벤치마크 리포트 출시
- 프랜차이즈·대기업 B2B 확장

## 4. 가격 전략 (Pricing Strategy)

### 4.1 구독 플랜

| 플랜 | 가격 | 주요 기능 | 타겟 |
|------|------|-----------|------|
| **Free** | ₩0 | 5분 매뉴얼<br>기본 근태<br>직원 5명까지 | 소규모 매장<br>신규 점주 |
| **Standard** | ₩29,000/월 | Free 기능 +<br>AI 법규 준수<br>기본 스케줄링<br>직원 20명까지 | 성장하는 매장 |
| **Premium** | ₩99,000/월 | Standard 기능 +<br>AI 수요 예측 스케줄링<br>인건비 효율 분석<br>무제한 직원 | 다점포 운영<br>효율 중시 점주 |
| **Best Practice** | ₩149,000~₩299,000/월 | Premium 기능 +<br>성공 매장 모델 템플릿<br>실시간 벤치마킹<br>전담 컨설팅 | 프리미엄 점주<br>프랜차이즈 |

### 4.2 가격 차별화 전략

**1. Value-based Pricing:**
- ROI 기반 가격 책정 (예: 인건비 10% 절감 → 월 ₩99,000 정당화)

**2. 성과 연동형 옵션:**
- Best Practice 플랜: 매출 증가/인건비 절감 성과에 따라 가격 조정

**3. 볼륨 디스카운트:**
- 다점포: 2점포 이상 시 20% 할인
- 프랜차이즈: 10점포 이상 시 별도 협의

## 5. 성장 플라이휠 (Growth Flywheel)

```
[Free Layer로 시장 확산]
         ↓
[데이터 수집 (성공 매장 패턴)]
         ↓
[AI 모델 학습·개선]
         ↓
[Best Practice 모델 고도화]
         ↓
[Subscription 가치 증대]
         ↓
[고객 락인 강화]
         ↓
[Matching Layer 활성화]
         ↓
[Network Effect 증폭]
         ↓
[Data Layer B2B 확장]
         ↓
[산업 표준화]
```

**핵심 메커니즘:**
- 사용할수록 데이터가 쌓이고
- AI 모델이 똑똑해지며
- 성공률이 높은 매장 모델을 자동 복제
- 신규 고객 유입 가속화

## 7. 요약

BestPractice HR SaaS는:
- **"기능을 파는 SaaS"**가 아닌 **"성공을 파는 SaaS"**
- 단순 근태 툴이 아닌 **"자영업 HR Operating System"**
- Freemium → Subscription → Matching → Data 다층 BM으로 **지속 성장 가능한 구조**

최종 목표:
**자영업 HR SaaS 시장의 De Facto Standard**
