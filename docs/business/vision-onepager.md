---

## 비전 (Vision)

허브스팟(HubSpot)이 인바운드 마케팅을 SaaS화한 것처럼,
우리는 **자영업 HR 베스트 프랙티스**를 SaaS화한다.

우리는 단순한 HR 관리 툴이 아니라,
**"성공하는 사장님들의 HR·운영 노하우를 시스템화한 AI 기반 경영 인프라"**를 구축한다.

이 플랫폼을 쓰는 순간, 누구든 **"성공하는 매장의 경영 공식"**을 자동으로 따르게 된다.

---

## 타겟 고객 (Target Users)

### 주요 타겟
- **소규모 오프라인 매장 점주** (카페, 음식점, 소매점 등)
- **매장 알바/파트타이머 직원**
- **노무사/세무사** 등 자영업 점주를 고객으로 둔 전문가 파트너

### 고객 페인 포인트
- 점주: 채용 실수, 인건비 과다, 직원 교육 부담, 법규 리스크
- 알바생: 불명확한 업무 기준, 경력 증명 어려움, 성장 기회 부재
- 파트너: 단발성 컨설팅의 한계, 지속 수익 모델 부재

---

## 핵심 가치 제안 (Core Value Proposition)

### 점주에게
- **데이터 기반 경영 컨설턴트**: 성공 매장의 HR 공식을 자동 적용
- 채용 실수 최소화, 인건비 최적화, 교육 시간 70% 절감

### 알바생에게
- **커리어 성장 코치**: 스킬 배지, 경력 리포트, 검증된 커리어 증명
- 단순 알바에서 성장하는 전문 인력으로 전환

### 파트너에게
- **지속 수익 채널**: 구독 수익 쉐어, 고객 유입 채널 제공
- 법률/세무 신뢰 기반 영업망 확대

## 핵심 전략: HR Best Practice Flywheel

```
[성공 매장] → [데이터 수집] → [AI 모델링] → [SaaS 상품화]
     ↑                                              ↓
[인재 성장/교육] ← [매칭/인증] ← [운영 효율 개선]
```

**플라이휠 작동 원리:**
1. 성공 매장의 운영 패턴 수집 (무료 매뉴얼·근무·학습 데이터)
2. AI가 공통 성공 요인 학습 (근무 스케줄, 교육 구조, 평가 기준)
3. 성공 모델을 SaaS 기능화 (AI 스케줄링, 매뉴얼 템플릿, 인재 매칭)
4. 신규 점주가 구독하여 성공 패턴 복제

## BM 구조 (5 Layer Model)

| 레이어 | 기능 | 수익 모델 |
|--------|------|-----------|
| ① Free Layer | 5분 매뉴얼, 기본 근태, 교육 자동화 | Freemium (데이터 확보) |
| ② Subscription Layer | AI 법규 준수, AI 스케줄링, 효율 분석 | Standard/Premium 구독 |
| ③ Best Practice Layer | "성공 매장 모델" 템플릿 SaaS | 프리미엄 구독 (성과 연동형) |
| ④ Matching Layer | 검증 인재 추천, AI 경력 코칭 | 채용 수수료 (고마진) |
| ⑤ Data Layer | 업종별 HR 지표 리포트, API | 데이터 라이선스 |

| 구분 | 기존 HR SaaS (샤플/시프티) | BestPractice HR SaaS |

| 브랜드 | 효율 툴 | 성공 복제 시스템 |
